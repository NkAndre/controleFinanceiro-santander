## SOBRE O PROJETO  
Este projeto foi criado para facilitar o acompanhamento de gastos e entradas diárias. O foco principal foi a fidelidade visual ao layout do banco Santander, utilizando a paleta de cores institucional e componentes customizados.

## ✨ Funcionalidades
Visualização de Saldo: Card principal com saldo atual dinâmico.

Ações Rápidas: Atalhos para Pagamentos, Transferências e Recargas.

Histórico Recente: Lista de transações com categorias (Lazer, Alimentação, etc.).

Gestão de Itens: Adição e remoção de transações com descrição e data.

Toggle de Privacidade: Opção para ocultar/exibir o saldo.
